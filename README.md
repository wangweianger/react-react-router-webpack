# react

>  * 使用知识点：
>  * react.js          轻量级mvvm框架
>  * webpack           前端自动话打包工具
>  * react-router      react路由
>  * react-redux       前端状态管理 
>  * babel             es6转译工具，用最前言的javascript做前端开发


## 项目步骤

1.安装node.js

2.安装项目依赖包

```
npm install
```

3.运行开发环境

```
npm run dev 
```

```
浏览器中访问：localhost:8000
```


4.打包生产文件

```
npm run build
```
